一、專題名稱

AI 心情日記分析師

利用人工智慧協助使用者紀錄與理解每日心情，提升自我覺察與心理健康。

二、製作動機

現代人生活壓力大，卻不常主動整理自己的情緒。本專題希望透過自然語言處理（NLP）技術，讓使用者在寫日記的同時，能自動獲得情緒回饋與可視化統計，促進心理健康與自我理解。

三、系統功能介紹

功能名稱 說明
日記輸入 使用者每日可輸入一段文字日記
情緒分析 使用 NLP 模型分析輸入文字屬於哪種情緒（快樂、悲傷、焦慮、平靜等）
圖表視覺化 生成情緒占比圖、週期變化圖
建議回饋 根據心情推薦正向語句或影片資源
匯出功能 將情緒記錄匯出為 CSV / PDF 供個人追蹤

四、系統架構圖

使用者 ➡ 輸入日記 ➡ NLP 模型分析 ➡ 分類結果 ➡ 圖表 & 建議 ➡ 儲存結果

前端：HTML/CSS + JavaScript + Chart.js

後端：Flask (Python)

AI 模型：BERT / DistilBERT 情緒分類模型（如：nlptown/bert-base-multilingual-uncased-sentiment）

五、使用技術

類別 技術
前端 HTML、CSS、JavaScript、Chart.js
後端 Python + Flask
AI 模型 Transformers（Hugging Face）、情緒分類模型
資料處理 pandas、NumPy
文字轉語音（選配） gTTS、pyttsx3

六、模型與資料來源

使用 Hugging Face 上公開的情緒分析模型（如 distilbert-base-uncased-finetuned-sst-2-english）

支援繁體中文可考慮：finbert、BERT Multilingual 或 fine-tuned 中文模型（需自行訓練或轉換）

資料儲存使用 SQLite 或 JSON 格式，保護個資

七、實作畫面示意

[📄 輸入日記] →「今天在捷運上看到一個小孩笑得好開心，讓我覺得很溫暖。」

→ AI 分析：情緒 = 快樂 😊

→ 顯示建議：「繼續保持這份好心情，祝你今天也愉快！」

八、預期效益

對象 收益
一般使用者 幫助自我反思與情緒追蹤
心理健康從業者 可作為輔助工具查看病人狀況變化
教育單位 / 學生 培養情緒覺察與寫作習慣

九、未來改進方向

  • 支援語音輸入與語音朗讀建議
  • 擴充情緒分類細緻度（例如 10 種以上）
  • 與 LINE Bot / Telegram Bot 整合，隨時記錄心情
  • 加入心理測驗功能（MBTI、情緒檢測問卷）
